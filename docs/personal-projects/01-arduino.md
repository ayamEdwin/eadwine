# Custom Arduino Uno R3

This project involved designing a clone of the Arduino Uno R3 board using KiCAD EDA. The goal was to apply PCB design and hardware engineering skills learned in coursework to build a reliable microcontroller development board.

## Project Highlights

- Designed the Arduino Uno R3 schematic and PCB layout in KiCAD.
- Applied best practices for power distribution, signal routing, and component placement.
- Focused on clear layout for the microcontroller, USB interface, power circuitry, and supporting headers.
- Created a board design suitable for future fabrication, even though the project remains at the design stage.

## Skills Developed

- PCB schematic capture and layout in KiCAD.
- Understanding of Arduino-compatible hardware architecture.
- Practical electronics design for embedded systems.
- Preparation for future hardware prototyping and small-scale manufacturing.

## Why it matters

This project strengthened my hardware design foundation and prepared me for future robotics projects that depend on compact, reliable embedded control boards.
