# Self-Balancing Robot Simulation (MATLAB)

This simulation project is rooted in my Control Systems I and II coursework. I built a dynamic MATLAB/Simscape model of a pendulum-style self-balancing robot to study system response and controller behavior.

## What I Built

- A mathematical model representing the dynamics of a self-balancing pendulum robot.
- Simulations using MATLAB with the Simscape multibody library to capture realistic motion and response.
- Control strategies that stabilized the system while keeping the pendulum upright under perturbations.

## Why It Matters

- Self-balancing robots are a classic control challenge that demonstrates the interaction between mechanical design and algorithms.
- This project helped me translate theoretical control concepts into a practical, simulated design environment.
- It reinforced the value of modeling, simulation, and iterative refinement before moving to hardware.

## Technical Skills

- MATLAB programming and Simscape model construction.
- Dynamic system analysis and response evaluation.
- PID control tuning and performance assessment.
- Simulation-driven design for robotics and automation.
