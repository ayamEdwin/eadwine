# Digital Clock
## Project Lead : Digital Clock Prototype – Year Two (2)
This digital clock was developed as a core assessment for the Product Development Course, demonstrating the full
lifecycle of creating an embedded system. Leading a team of five mechatronics students, I coordinated the firmware
development using the Mbed OS framework—specifically leveraging Timer and Ticker APIs—to program an NXP KL25Z
development board powered by an ARM Cortex-M0+ processor. My role encompassed end-to-end ownership, from
initial code architecture and task delegation to hands-on PCB soldering, system debugging, and iterative testing. We
successfully transformed the initial concept into a fully functional product prototype, navigating challenges such as
electrical faults and firmware timing issues. This project served as a practical application of embedded programming,
hardware integration, and agile problem-solving in a team environment.
